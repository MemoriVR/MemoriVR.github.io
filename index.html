What are Memory Palaces?

Memory palaces link visualization of spatial environments to information that we want to recall. It specifically targets two areas of the brain. The dorsolateral prefrontal cortex (DLPFC) and the Medial Prefrontal Cortex (MPFC).

Why Memory palaces?

This graph, from Dresler's study, shows the difference in participant's memory retention after different time periods. With mnemonic training (Loci method), participants showed drastically better results than the other two control groups.

What does VR benefit Memory palaces?

Memory palaces heavily deals with visualization and spatial awareness. The best technology to enhance this experience in VR. With VR, the user would be able to have a tangible version of their memory palaces. This would make it easier for some people to use Loci, and make the experience even better for others.
